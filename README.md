# Global Developer Trends: Exploring Current and Future Technology Preferences

A data analysis project examining global developer technology preferences using the Stack Overflow Developer Survey to identify current usage patterns and emerging trends in programming languages and databases. This is my capstone project for the IBM Data Analyst Profesisonal Certificate. 

## üìä Project Overview

This project analyzes the Stack Overflow Developer Survey dataset to compare the technologies developers use today with those they want to use in the near future. The insights help data professionals, hiring managers, and educators align their skills and strategies with global technology trends.

**Key Questions Explored:**
- Which programming languages and databases are most widely used today?
- What technologies are developers most interested in learning next year?
- How can professionals position themselves for future opportunities?

## üîç Key Findings

### Programming Languages
- **Python and JavaScript** dominate current usage and remain the most desired for next year
- **Emerging interest**: TypeScript, Go, and Rust show strong growth despite lower current usage
- **Stable demand**: Java and C maintain steady interest in enterprise and legacy systems

### Databases
- **SQL databases** (particularly PostgreSQL) remain dominant in current usage and future interest
- **Rising stars**: NoSQL and cloud databases like MongoDB and Redis show higher projected growth
- **Traditional stability**: MySQL, Oracle, and SQL Server maintain consistent usage with slower growth

### Implications and Recommendations for Data Professionals
- Prioritize learning Python, while embracing continuous learning of modern languages (TypeScript, Go) to stay relevant in the job market.
- Develop skills in both relational (SQL, PostgreSQL) and NoSQL/cloud databases
